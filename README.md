# ⚡ Reaction Tester

Быстрый и точный тестер времени реакции с красивым киберпанк-дизайном.

## 🎮 Режимы

### CLASSIC Mode
Классический режим тестирования. Жди зелёного квадрата и нажимай как можно быстрее!

### INSANE Mode
Продвинутый режим. Квадрат появляется в случайных местах экрана. Попробуй поймать его!

## ✨ Особенности

- ⚡ **Точное измерение** - реакция измеряется в миллисекундах
- 📱 **PWA (Progressive Web App)** - работает офлайн и установляется как приложение
- 📊 **Статистика** - отслеживание результатов: среднее, лучший и худший результат
- 🎨 **Киберпанк дизайн** - темная тема с неоновыми цветами
- 📦 **Кроссплатформа** - работает на всех устройствах (веб)
- 🐍 **Python версия** - десктопное приложение на tkinter

## 🚀 Быстрый старт

### Веб-версия
1. Откройте `index.html` в браузере
2. Выберите режим (CLASSIC или INSANE)
3. Нажимайте как можно быстрее!

**Установка как PWA:**
- На Android: Откройте меню браузера → "Установить приложение"
- На iOS: Нажмите Поделиться → "На главный экран"

### Python-версия
```bash
python3 reaction_timer.py
```

Требования: Python 3.6+

## 🎯 Оценки

- 🔥 **Невероятная** - < 150 мс
- ⭐ **Отличная** - 150-250 мс
- 👍 **Хорошая** - 250-350 мс
- 💤 **Неплохая** - > 350 мс
