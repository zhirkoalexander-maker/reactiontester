# ⚡ Reaction Tester

Комплексный пакет тестов для проверки времени реакции с красивым киберпанк-дизайном.

## 🎮 Режимы Тестирования

### 1️⃣ CLASSIC Mode
Классический режим тестирования реакции. 
- **Задача:** Жди зелёного квадрата и щёлкай как можно быстрее
- **Сложность:** Фиксированная
- **Идеально для:** Базовой оценки времени реакции

### 2️⃣ INSANE Mode
Продвинутый режим с движущейся целью.
- **Задача:** Квадрат появляется в случайных местах экрана
- **Сложность:** Фиксированная
- **Идеально для:** Тренировки и усложнённого тестирования

### 3️⃣ VISUAL Mode (Новый!)
Тест зрительной реакции - найди лишнюю фигуру.
- **Задача:** Из 4 квадратов найди один отличающейся
- **Сложность:** 3 уровня (Easy/Medium/Hard)
  - **Easy:** Различные цвета
  - **Medium:** Различные формы
  - **Hard:** Абстрактные паттерны
- **Идеально для:** Проверки зрительного восприятия и аналитических способностей

### 4️⃣ MENTAL Mode (Новый!)
Тест умственной реакции - реши математические примеры.
- **Задача:** Решай примеры и выбирай правильный ответ как можно быстрее
- **Сложность:** 3 уровня (Easy/Medium/Hard)
  - **Easy:** Простые примеры (1-10) с + и -
  - **Medium:** Средние примеры (1-20) с +, - и ×
  - **Hard:** Сложные примеры (1-50) с +, -, ×, ÷
- **Идеально для:** Проверки концентрации и логического мышления

## ✨ Особенности

- ⚡ **Точное измерение** - реакция измеряется в миллисекундах
- 📁 **4 разных типа тестов** - оцени разные аспекты реакции
- 🎚️ **Регулируемая сложность** - от простого к сложному
- 📱 **PWA (Progressive Web App)** - работает офлайн и установляется как приложение
- 📊 **Подробная статистика** - среднее, лучший, худший результат и точность
- 🎨 **Киберпанк дизайн** - тёмная тема с неоновыми цветами
- 📦 **Кроссплатформа** - веб + десктопный Python

## 📊 Статистика

После каждого теста видишь:
- **Время реакции** - сколько миллисекунд ушло на ответ
- **Оценка** - качественная оценка результата
- **Попытки** - количество пройденных тестов
- **Среднее** - среднее время за все попытки (CLASSIC/INSANE)
- **Точность** - процент правильных ответов (VISUAL/MENTAL)
