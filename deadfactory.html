>> REACTION TEST MODES <<<
01 // CLASSIC
The basics. Wait for the green signal to appear, then click. Simple reaction test.
EASY: 1-2s wait • NORMAL: 1.5-2.5s wait • HARD: 2-3.5s wait
02 // INSANE
The green square moves. Click it before it jumps to a new position. Each wrong click = retrying.
EASY: 0.8-1.5s delay • NORMAL: 0.6-1.2s delay • HARD: 0.4-1s delay
03 // VISUAL
Find the odd emoji (🔵) among the circles (⭕). Beat the time limit.
EASY: 6 emojis, 15s limit • NORMAL: 9 emojis, 10s limit • HARD: 12 emojis, 6s limit
04 // MENTAL
Math problems. Pick the correct answer from 4 options. Beat the time limit.
EASY: Basic math, 20s limit • NORMAL: Multiply & squares, 10s limit • HARD: Complex, 5s limit
05 // MEMORY
Remember the sequence, then repeat it. Longer sequences on harder difficulties.
EASY: 3 digits, 3s display • NORMAL: 4 digits, 2s display • HARD: 5 digits, 1s display
• Each test has 3 ATTEMPTS per session
• Results are saved automatically (localStorage)
• Change difficulty before starting
• Best scores show on main screen
• Correct/success rate is tracked
