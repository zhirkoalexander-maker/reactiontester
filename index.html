A Beginner's Project
Learning web development with JavaScript. This is a practice project built from scratch.

All tests use keyboard & mouse. Tests track scores locally. Each test = 3 attempts.
Test Modes
01 Classic
Wait for green signal. Click as fast as you can.
Easy: 1-2s wait | Normal: 1.5-2.5s wait | Hard: 2-3.5s wait
02 Insane
Click the moving green square before it jumps.
Easy: 0.8-1.5s delay | Normal: 0.6-1.2s delay | Hard: 0.4-1s delay
03 Visual
Find 🔵 among ⭕ circles. Beat the time limit.
Easy: 6 items, 15s | Normal: 9 items, 10s | Hard: 12 items, 6s
04 Mental
Solve math problems from 4 answer options.
Easy: Basic math, 20s | Normal: Multiply/squares, 10s | Hard: Complex, 5s
05 Memory
Remember sequence and repeat it. Use 1-9 keys.
Easy: 3 digits, 3s display | Normal: 4 digits, 2s | Hard: 5 digits, 1s
• Keyboard: Space (Classic), 1-9 (Memory), Enter (Chat)
• Scores saved to browser storage
• ESC closes any test window
